Преобразование типов в JavaScript

Задание 3.

Когда мы пытаемся преобразовать к другим типам данных не совсем стандартные сущности, то иногда получаются необычные значения. Например, если мы преобразуем к строке функцию alert(), то увидим следующее:
console.log(String(alert));
// Возвращает function alert() { [native code] }

Задача состоит в том, чтобы преобразовать явно к string, number и boolean следующие сущности, посмотреть на результат и проанализировать его:

  1. console.log
  2. { name: 'Maxim' }
  3. Symbol('key')
  4. Number
  5. ''
  6. 0
  7. -10
  8. '-105'
